Sobre mim

Meu nome é Naiara Suelen Correa, sou estudando de Análise e Desenvolvimento de Sistema na Faculdade de Tecnologia de Sorocaba-FATEC. Meu primeiro contato com a programação foi em 2019 quando resolvi procurar um curso para fazer, desde entao me apaixonei pela programação e venho seguindo estudando para enfrenter os desafios.

Comecei a trabalhar na área em 2022 como estágiaria de desenvolvimento back-end com a linguagem Java. Sou um pouco curiosa e isso me faz estar sempre tentando aprendendo coisas novas.

    Meus conhecimentos

  • Linguagem de Marcação de Hipertexto — o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.
  • CSS é a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código criado por linguagens como HTML, XML ou XHTML, por exemplo.
  • O JavaScript é amplamente categorizado como uma linguagem de criação de scripts ou uma linguagem interpretada. Permite criarmos páginas web cada vez mais dinâmicas e interativas
  • Java é uma linguagem de programação orientada a objetos desenvolvida na década de 90.
  • MySQL é um sistema de gerenciamento de banco de dados relacional de código aberto (RDBMS) apoiado pela Oracle e baseado em linguagem de consulta estruturada (SQL).
  • O Kotlin é uma linguagem de programação de código aberto estática, que oferece suporte para programação orientada a objetos e funcional. Ele fornece sintaxe e conceitos semelhantes de outras linguagens, incluindo C#, Java e Scala, entre muitas outras.
